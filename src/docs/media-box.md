## API

| 参数      | 说明          | 类型      | 可选值                           | 默认值  |
|---------- |-------------- |---------- |--------------------------------  |-------- |
| title | 卡片主题名称| String | — | - |
| actions  | 按钮配置列表  | Array   | — | - |
| src | 按钮图标链接| String   | — | - |

### Actions配置信息
| 参数      | 说明    | 类型      | 可选值       | 默认值   |
|---------- |-------- |---------- |-------------  |-------- |
| icon    | 菜单按钮图标   | String    | — | —    |
| text     | 菜单按钮描述   | string  |  -          |    —     |
| key     | 方法触发key  | string    |   - |    -  |
